The problem

Real cameras don't give you clean photos.

Benchmarks train on pristine images. The real world hands you motion blur, sensor noise, compression artefacts and low light — frequently stacked on the same frame. We model that head-on with six physically-grounded degradations, composed at random.

〰️

Motion blur

Linear streak from camera / subject movement.

⊙

Defocus blur

Lens disk / bokeh from a mis-focused lens.

▦

Gaussian blur

Smooth low-pass softening.

⁘

Sensor noise

Broadband high-frequency grain.

▤

JPEG artefacts

Block-frequency compression mush.

◐

Low light

Gamma + gain under-exposure.

FFT spectral module

Degradation lives in the spectrum — blur removes high frequencies, noise adds them. A learnable per-frequency complex filter gives the network a global receptive field a normal conv can't reach.

rfft2 → learnable gain → irfft2 → residual
🎯

SimOTA assignment

Instead of one-ground-truth-one-anchor, build a cost matrix (class + IoU) and let each object claim a dynamic number of positives. Better samples, better gradients.

cost = cls + 3·IoU → dynamic-k
📦

IoU-guided loss

The objectness target is the actual IoU of the predicted box, so confidence reflects localisation quality — and NMS keeps the boxes that are truly well-placed.

obj_target = IoU(pred, gt)
